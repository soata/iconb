# iconb

easily change application's icon.

## Features

Change application's icon with .icns in a directory.

### Applications

- [appname].icns (like "Google Chrome.icns")

### Finder

- finder/finder.png
- finder/finder2.png

### Calendar

- calendar/App.icns
- calendar/App-empty.icns

## Quickstart

```bash
$ brew install iconb
$ iconb run {path}
```
